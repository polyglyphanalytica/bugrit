# **App Name**: Bugrit

## Core Features:

- Input Capture: Capture user input as a keyword or topic.
- Firestore Integration: Store user input and generated content metadata in Firestore, under a user's collection.
- Gemini Flash Paragraph Generation: Apply the user's keyword or topic to Gemini Flash, and generate a paragraph of text content; provide the LLM with a tool to determine if the content of a website is worth including in its output.
- Image Generation: Generate an image based on the user's keyword or topic using AI.
- Image Storage: Store the generated image in a user-level storage bucket.
- Content Display: Display the generated paragraph and image in a user-friendly format.

## Style Guidelines:

- Primary color: Light lavender (#D0BFFF) to inspire creativity and imagination.
- Background color: Off-white (#F8F8FF) for a clean and unobtrusive canvas.
- Accent color: Soft pink (#FFB6C1) for highlights and calls to action.
- Font pairing: 'Poppins' (sans-serif) for headlines and short amounts of text, combined with 'PT Sans' (sans-serif) for body text. This provides a clean, modern, and readable feel.
- Use simple, outlined icons to represent different content types or actions.
- Use a clean and responsive layout to ensure the app is usable on all devices.
- Add subtle animations and transitions to create a more engaging user experience.
